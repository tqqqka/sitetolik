TOLY SUMMER static site

Как открыть:
1. Открой index.html в браузере.
2. Для GitHub Pages загрузи все файлы в репозиторий.
3. Если есть фото артиста, создай папку assets и положи туда файл:
   assets/toly-summer.jpg

Страницы:
- index.html — главная
- about.html — об артисте
- music.html — музыка
- video.html — видео
- contacts.html — контакты

В коде нет упоминаний стороннего конструктора.



- После распаковки создай/открой папку fonts и положи туда файл:
  fonts/YSMusic-HeadlineBold.ttf

Шрифт:
CSS настроен так, чтобы шрифт применялся ко всему сайту.
После распаковки положи файл YSMusic-HeadlineBold.ttf в одну из этих папок:
1) toly_summer_site/fonts/YSMusic-HeadlineBold.ttf
или
2) toly_summer_site/YSMusic-HeadlineBold.ttf

Без самого .ttf файла браузер покажет запасной Arial.

- отключён правый клик;
